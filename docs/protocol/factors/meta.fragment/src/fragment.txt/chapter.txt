A factor referring to an object and an internal element or a relative reference to an element
without an explicit object context. An abstract form of &<meta.element>; where &<meta.element>
refers concretely to Elements within Factors, Fragments intend to be conceptually independent.

Currently, this factor type is unused in fault.io projects and only serves to identify the concept.
