sequencefunction

sequence(directory, nodes)

Generator maintaining the state of the sequencing of a traversed dependency graph. This generator emits factors as they are ready to be processed and receives factors that have completed processing.

When a set of dependencies has been processed, they should be sent to the generator as a collection; the generator identifies whether another set of modules can be processed based on the completed set.

Completion is an abstract notion, sequence has no requirements on the semantics of completion and its effects; it merely communicates what can now be processed based completion state.
