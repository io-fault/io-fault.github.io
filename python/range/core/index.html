Critical functions supporting range types.

scalefunction

scale(n, target, origin, divmod)

Given a number, target range, and a origin range. Project the number onto the target range so that the projected number is proportional to the target range with respect to the number's relative position in the origin range. For instance::

>>> scale(5, target=(1,100), origin=(1,10)) (50, 0, 9)

The return is a triple: (N, remainder, origin[1] - origin[0]) Where the second item is the remainder with the difference between the source range's end and beginning.
