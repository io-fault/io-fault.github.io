Project Directory Protocol for explicit factor specifications.

Explicitly Typed Factors is a project factor protocol designed for projects that primarily consist of factors that are composed of single source files. Notably, languages such as Interpreted Lisps and Python are likely users of explicit.

Composites are designated as directories containing a src directory and a factor.txt file. When present, query will associate the tree contained in the source directory as the sources composing the factor whose type is designted in the, previously mentioned, text file. All other directories are perceived to contain Whole Factors whose domain is inferred from the file's identified language.
