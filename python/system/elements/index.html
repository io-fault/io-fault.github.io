fault.system.elementshttp://if.fault.io/factors/python.module

Interfaces and declarations for managing interpretation contexts for element types.

Elements are instances of often used conceptual types such as dates, timestamps, and paragraphs. Within a given application, it is reasonable that consistency be desired among the implementation types for elements. In cases where foreign interfaces have been connected, it is likely critical that some desired set be expressed so that data with a proper interface can be received.
