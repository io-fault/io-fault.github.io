Monitorconfiguremethod

configure(self, lines)

Configure the scrolling region allocating lines at the top or bottom of the screen for status display.

This method should be called after window changes of any type. The window size is refreshed from the device; monitors should also be reallocated so that their Context positions can be adjusted.

setupfunctionMonitor

setup()

Initialize the terminal for use with a scrolling region.

The given lines determines the horizontal area allocation to manage at the bottom or top of the screen. If negative, the top of the screen will be allocated. Allocating both is not supported by this interface as Monitor only manages one fault.terminal.matrix.Context.
