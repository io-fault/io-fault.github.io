Frame drawing kit for delimiting areas of display.

Frames are drawn from pattern expressions; the expressions are formed from a sequence of intersections that designate the special cases found along the primary line pattern.

render_verticalfunction

render_vertical(canvas, palette, expression)

Interpret the expression in the context of the canvas yielding the events necessary to cause the frame borders to be displayed.

render_horizontalfunction

render_horizontal(canvas, palette, expression)

Interpret the expression in the context of the canvas yielding the events necessary to cause the frame borders to be displayed.

render_boxfunction

render_box(canvas, corners, hpalette, vpalette, width, height)

Draw the border of a box with the given width and height. The width and height are the inclusive dimensions of the contents; the frame will be drawn around that area.

The yielded commands are entirely relative and can be cached for replays. This should only be used for raw mode displays.
