ictlchrfunctionint

ictlchr(ctlid)

Transform a 'control' fault.terminal.types.Event.identity into its literal form.

When a control character is transformed into a fault.terminal.types.Event, its identity is processed into a lowercase english alphabet character consistent with the usual visualization.

This function inverts the process for applications needing the actual target control character. Notably, relying on the fault.terminal.types.Event.string is inappropriate in the case where the character was escaped and the event is qualified with some modifiers.

Parse, normally CSI, returning the intermediates, parameters, terminator, and any following text as a remainder.
