Foreground management controller for main thead.

For system based processes, it is common to spawn threads to perform and manage work while the main thread is managed with meta.system.library.protect. In the case of an interactive shell, the foreground needs special attention in order to respond to job suspension events. foreground provides the necessary mechanics for signalling foreground completion to an meta.io based shell.
