Positionclass

object

Mutable position state for managing the position of a cursor with respect to a range. No constraints are enforced and position coherency is considered subjective.

PositionProperties

datum

The absolute position. (start)

offset

The actual position relative to the datum. (current=datum+offset)

magnitude

The size of the range relative to the datum. (stop=datum+magnitude)

Positionzeromethod

zero(self)

Zero the offset and magnitude of the position.

The datum is not changed.

Positionmovemethod

move(self)

Move the position relatively or absolutely.

Perspective is like the whence parameter, but uses slightly different values.

-1 is used to move the offset relative from the end. +1 is used to move the offset relative to the beginning. Zero moves the offset relatively with update.

Positionconstrainmethod

constrain(self)

Adjust the offset to be within the bounds of the magnitude. Returns the change in position; positive values means that the magnitude was being exceeded and negative values mean that the minimum was being exceeded.

Positioncollapsemethod

collapse(self)

Move the origin to the position of the offset and zero out magnitude.

Positionnormalizemethod

normalize(self)

Relocate the origin, datum, to the offset and zero the magnitude and offset.

Positionrepositionmethod

reposition(self)

Reposition the datum such that offset will be equal to the given parameter.

The magnitude is untouched. The change to the origin, datum, is returned.

Positionstartmethod

start(self)

Start the position by adjusting the datum to match the position of the offset. The magnitude will also be adjust to maintain its position.

Positionbisectmethod

bisect(self)

Place the position in the middle of the start and stop positions.

Positionhaltmethod

halt(self)

Halt the position by adjusting the magnitude to match the position of the offset.

Positioninvertmethod

invert(self)

Invert the position; causes the direction to change.

Positionpagemethod

page(self)

Adjust the position's datum to be at the magnitude's position according to the given quantity. Essentially, this is used to "page" the position; a given quantity selects how far forward or backwards the origin is sent.

Positioncontractmethod

contract(self, offset, quantity)

Adjust, decrease, the magnitude relative to a particular offset.

Positionchangedmethod

changed(self, offset, quantity)

Adjust the position to accomodate for a change that occurred to the reference space--insertion or removal.

Similar to contract, but attempts to maintain offset when possible, and takes an absolute offset instead of a relative one.

Positiondilatemethod

dilate(self, offset, quantity)

Adjust, increase, the magnitude relative to a particular offset.

Positionexpanddata

expand = dilate

Positionrelationmethod

relation(self)

Return the relation of the offset to the datum and the magnitude.

Positioncompensatemethod

compensate(self)

If the position lay outside of the range, relocate the start or stop to be on position.

Positionslicemethod

slice(self)

Construct a slice object that represents the range.

Vectorclass

object

A pair of Position instances describing an area and point on a two dimensional plane.

Primarily this exists to provide methods that will often be used simultaneously on the vertical and horizontal positions. State snapshots and restoration being common or likely.

Cacheclass

object

Mapping interface for user trans-refraction communication. (Local Clipboard)

Maintains a set of slots for storing a sequence of typed objects; the latest item in the slot being the default. A sequence is used in order to maintain a history of cached objects. The configured limit restricts the number recalled.

Console clipboard.
