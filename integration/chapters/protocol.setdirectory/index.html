selectfunction

select(nodes, index)

Given the contents of an element, extract the properties from the set located at the designated index. Normally, properties are expected to be presented at the beginning of some content and, therefore, index defaults to 1.

If the element at index is not a 'set', or nodes is an empty, an empty iterator will be returned.

Error conditions produced during the production of a property set indicate that it is not a property set at all and will cause an empty list to be returned.
